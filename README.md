# lispf_ck-compiler
 repository for lisp_f_ck compiler activity, continued work from https://github.com/fabiolamfleury/lispf_ck-interpreter

 This project converts lispfuck code to brainfuck. It supports the use of functions, to use it select your file.lf and name your .bf file according to this command:

 $ python3 lispf_ck.py examples/add_ask.lf -o file_name.bf
